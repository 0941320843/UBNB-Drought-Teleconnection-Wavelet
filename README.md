# UBNB-Drought-Teleconnection-Wavelet
Data for drought teleconnection analysis over Upper Blue Nile Basin (1987–2018)

Data Availability
This repository contains the data used in the article:
"Investigating the Drought Teleconnections of Upper Blue Nile Basin, Ethiopia Using Wavelet Coherence"
 Data description
- Rainfall Anomaly Index (RAI) from 20 stations (1987–2018)
- Climate indices: Niño4, Niño3.4, IOD, ISMI
 Format
All data are provided in CSV format.
